# Parkrun Images

Place images for each parkrun in this directory with the following filenames:

1. `southwick-country-park.jpg` - Southwick Country Park parkrun
2. `brickfields-park.jpg` - Brickfields Park parkrun
3. `chippenham.jpg` - Chippenham parkrun
4. `lydiard.jpg` - Lydiard parkrun
5. `marlborough-common.jpg` - Marlborough Common parkrun
6. `melksham.jpg` - Melksham parkrun
7. `salisbury.jpg` - Salisbury parkrun
8. `seven-fields.jpg` - Seven Fields parkrun
9. `bath-skyline.jpg` - Bath Skyline parkrun
10. `thoulstone.jpg` - Thoulstone parkrun
11. `tidworth.jpg` - Tidworth parkrun
12. `quakers-walk.jpg` - Quaker's Walk parkrun

## Image Recommendations

- **Format**: JPG or PNG
- **Size**: Recommended 800x600px or larger (will be cropped/covered)
- **Aspect Ratio**: Landscape orientation works best
- **Content**: Photos of the parkrun location, course, or scenery

The images will be displayed as background images on each calendar month tile with a green overlay for text readability.
